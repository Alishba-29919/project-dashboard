# 🚀 Advanced React Admin Dashboard

Ek modern, responsive aur secure admin dashboard jo React aur Tailwind CSS se banaya gaya hai. Is mein Dark Theme aur Orange accents ka use kiya gaya hai jo ise kaafi premium look deta hai.

## ✨ Features
- **Secure Authentication:** Login aur Logout ka mukammal system.
- **Protected Routes:** Sirf authenticated users hi dashboard access kar sakte hain.
- **Dynamic Sidebar:** Navigation ke liye smooth aur interactive sidebar.
- **Responsive Layout:** Mobile aur Desktop dono par behtareen chalta hai.
- **Task Management:** Tasks aur orders ko manage karne ke liye alag pages.
